# Roadmap

This roadmap is not a promise of delivery. It documents likely directions for the Free / Community version and possible Pro-level features.

## Free / Community

The Free / Community edition is intended to be a working NinjaTrader 8 tool, not a demo or a limited preview.

Current Free / Community scope:

- Working chart-based risk/reward panel.
- Manual OP/SL/TP interaction.
- 1TP, 2TP, and 3TP planning.
- Bracket execution from the panel.
- OCO protection after entry fill.
- BE and CLOSE controls.
- Instrument presets.
- Chart Trader account detection.
- Documentation and safety notes.
- Bug fixes for supported NinjaTrader 8 workflows.

## Candidate Improvements

- Better visual order-state feedback on the panel.
- More robust account detection fallback.
- Partial fill handling improvements.
- Optional confirmation before live order submission.
- Export/import panel presets.
- More language polish.
- More screenshots and examples.

## Possible Pro Direction

These features may belong in a separate Pro edition:

- risk-based position sizing,
- account balance percentage sizing,
- daily loss guard,
- max contracts guard,
- max trades per session,
- session time lockout,
- hotkeys,
- auto BE after TP1,
- trailing stop modes,
- partial close buttons,
- instrument profile manager,
- compiled installer,
- support and update channel.
