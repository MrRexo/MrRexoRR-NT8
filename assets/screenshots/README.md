# Screenshots

Place public-facing screenshots in this folder.

Recommended format:

- PNG
- 1400-1800 px wide for full chart screenshots
- 800-1200 px wide for focused panel screenshots
- dark chart theme preferred for consistency
- no account numbers, balances, personal information, broker IDs, or prop firm identifiers

Recommended filenames:

- `01-panel-overview.png`
- `02-multi-tp-mode.png`
- `03-order-toolbar.png`
- `04-bracket-orders.png`
- `05-break-even-close.png`

See `docs/screenshots.md` for the capture scenario.
